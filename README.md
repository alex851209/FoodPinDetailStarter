# FoodPinDetailStarter
FoodPin是一款餐廳評論app，可以讓使用者紀錄到訪過的餐廳，除了餐廳名稱、電話及地址外，也可以留下綜合評價以供下次參考。


空白首頁，可點右上方新增餐廳
